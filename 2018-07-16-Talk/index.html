对话

0

流水消逝于泥缝间。

你是否曾对自己不得体的发言多有懊恼？ 那些面红耳赤，忿怨不平的怒火？ 那些想要袒露，又担忧放弃的羞涩？ 那些会上无言，会后思涌的悔悟？

人思如泉涌。人们渴望共鸣俱流却多分歧争端，气愈急心愈狭，僵入死局； 泉水欲奔流入海却往往泥沙分流，量愈小流愈慢，直至干涸。人无共识意，江无聚海流。

思维无迹可寻，只能对话交流中得窥一二。人类需要共识，却天生固执己见。上到国家对话，下到家庭谈话，异己纷争随处可见，知交之谈却难觅影踪。戴维·伯姆在1970年对于沟通的观察依然适用于今日：

过去几十年来，现代科技发展一日千里，广播，电视与飞机，卫星构成一个庞大的网络联通全球，全世界的任何一个角落都几乎可以在瞬时间彼此联相连，然而与此相对应的，则是人与人之间感觉日渐隔阂，彼此间的沟通以空前的速度，变得每况愈下。

如今多了互联网与手机，状况有好转吗？ 那些为方寸屏幕低下的头颅何时能够抬起？ 网络言语暴力似乎更比现实过犹不及？ 时间流逝着，人们貌近神离，相俱无言，更甚者直诉暴力。

人心各异，其间不过几尺，却如远山相隔万里，千泉各自暗涌，何能共聚入海流？

千流竞涌

对话是人与人间意义之流淌　(flow of meaning)。

水有江河溪海，话语又有何面貌？

我们每天都在说话，但当真明白自己所言何物？ 表达说话的词似是而非： 会谈，谈心，聊天，谈判，讨论，争辩，劝告，商量。 形容交流的词也是千奇百怪： 众口铄金，言不由衷，答非所问，左右言它，洗耳恭听，话中有话，绵里藏针。王松涛先生将不同谈话形式归结到至少以下四型：

  • 聊 ，东拉西扯，聊字本意耳鸣，徒然耳旁嗡嗡之声。
  • 辩 ，相互竞争，证明你对我错，以达成我赢你输结果。
  • 商 ，相互妥协，无论对错真理，彼此折中而达成满意结果。
  • 谈 ，人人认真倾听他人，人人表达内心真实看法，为求真理以验而言。

聊者，重倾泻。觥筹交错，兴高采烈，好不快活。但无论如何眉飞色舞，神侃狂聊过后，空虚怅然滋长； 辩者，重输赢。输家愤懑不平，赢家自鸣得意，看似交锋激荡，依然各自固我：盲人摸象，各得腿尾，永无俱得真象之日；商者，重谐乐，无关真理，面子给足即可。如地表浅垄相隔，互瓢些水以致意，地下却岩结千层，永无相通之日，乡愿而已。

真正能汇流成海的，是谈，是意义的相互流动： 在最自由，最诚挚的交流中涌现真知灼见。 这样的交流弥足珍贵，虽常如昙花，但一有所尝，终身难忘。当你与挚友推心置腹，开诚布公，当你放下我执真正思考他人想法，当你勇敢说出真实想法得到回应，凡此种种时刻，人如银鲤跃过龙门，真正感受共识壮阔。

涛间汹涌， 风云开阖。

浪遏飞舟

既然自由对话如此可贵， 为何真诚交流如此稀有？ 是什么阻碍了人们真诚交流？

根源在于人之思维。 奔流的思维之泉，从源头开始就犯了错误，自己卷起泥沙堵死了通往大海的路而不自知。

人类思维总在分裂地看世界。 戴维·伯姆以「碎裂的钟表」作喻： 原本手表的零部件存在内在有机联系，而碎片毫无联系可言。人们常常将原本并不独立的事物还原得更支离破碎，如此方便了人类认知，却无法觉察事物整体内在联系。如此，人与人的观念就永远失去调和乃至超越的可能。世界上个体间，集体间，价值体系间，信仰间，国家间等等根本矛盾皆在于此。

根本矛盾易发冲突，刺激人们自我防备的本能。 每个人都有自己根深蒂固的不思维假定，人们往往把自己想法当作自己。因此当想法不被接受，就感觉自己受到攻击，丛林法则下对身体的自我保护本能同样发生在了思维上，即使当前并非丛林险境。所谓「脑子一热」： 一方面当你一味攻击对方，便不会仔细思考对方所言是否可取，只为攻击他人而争吵，交流自然无以为继；另一方面这种险境压力要求思维更好地表达自我形象，自欺毒脓便于此鼓包。

最致命的是，人们对思维过程难以自觉。 人们同时认为自我想法就是现实本身，真理本身，而非需要纠偏的表征。人们可以感受到自己的手，可以根据不同需要变换手形，但却难以对自我思维如此。思维可以探查万物，却不知道自己做了什么。思维本身导致错误，却认为责不在己。错误思维源源不断产生错误想法，事情往往因此变得更糟。

自我是多层演化堆积的结果，个人思维是集体思维的结果。你所想当然的自我思想，其实只是基因，父母，集体为你编制的「人生脚本」的运行之果，而你难以自觉。当你无法承认心力有限而世事复杂，无法辨识想法，自我与现实三者之别，无法放下自我狭隘碎片，悬置自我思维过程，对话不复存在，自由无处可寻。

清泉高歌奔入海，势必要破开那亿万年沉积之尘岩。

中流击石

对话困难重重，如何能够克服？ 我们如何进行对话？ 又需要注意什么？

对话是什么，又不是什么？ 对话是人与人直接面对的过程，不是理论探究或苦思冥想； 对话也不是心理辅导，虽然长期对话对心理障碍有一定功效； 对话亦不是诉苦大会，虽然不可避免会涉及个人问题，但这并不是信任与真知灼见的真正来源。对话的唯一要义在于： 破除固执己见之绝对，涌生共识共创之必要。

开始对话前需探讨对话本身。 当我们召集人们开启对话，需要预先讨论： 我们为什么要对话？ 意义是什么？有哪些需要注意的地方？ 知晓行动本身价值而自发尝试，相较被动接受别人安排尝试，云泥之别。就如费曼在自传中提及那些为曼哈顿计划作计算的年轻人，被告知自己所参与计划的伟大意义后，表现出了令人吃惊的效率与创造力。勾起对话者的对话兴趣，我们才能更好地对话。

关注对话过程而非身份认同感。 亲疏地位之别会扭曲人对实际过程的认知，成为阻碍： 你对我不屑一顾，我对你嗤之以鼻。一场对话之成功不在于人们有多亲近，而在于大家关注点是否都在对话本身上。倾听，观察，关注思维发生过程，观察其紊乱所在，辨别出错原因与影响因素，就事论事。只有悬置己见，你才能真正看清思维运作过程，看见真知真正流向。

对话需定期长期，1到2年以上为宜。 当一个对话团体刚刚成立时，人们常常会东拉西扯，避免尴尬。所谓「江湖险恶」的流行人际学影响下，人们不愿直面任何事情，触及任何深层问题。开始时更多是协商，寻求大家都愿意接受的对话方式，此让对话继续下去。人们在对话中会受挫，暴躁，厌倦，激动，胶着…… 只有认真长期进行，破除种种障碍，成员各自深层思维假定自会浮现。 困难总伴惊喜，只是这种惊喜并非线性，需要耐心引导涌现。

万事开头难，实际困难种种,能道者不过二三。唯一可以肯定的是：开始尝试! 不懈改进! 滴水而石穿,百川汇海自非一日之功。

乘风破浪会有时，直挂云帆济沧海。

惊涛拍岸

现实中真诚对话如行万方沙漠，寻一眼绿洲。艰难如此，甘澧何在？

对话体贯穿古今东西： 西方《理想国》苏格拉底与柏拉图对谈，东方《论语》 孔子师生对曰，佛道机锋禅语，皆由对话示道。而现代对话理论则首推伯姆对话(Bohmian Dialogue)。 对话不仅解决交流障碍，亦可求意识本质行径。通过关注对话过程，进行长期对话来促使人们看见自己的思维假定，理解他人的思维假定，共创超越对立矛盾的新见解，从而有望消解人与人日益严重的隔阂与分歧。

伯姆心忧人类思维无迹可寻的紊乱乃祸乱之源，自1970年便开始思考对话相关问题，直至1992年去世都在完善自己的对话理论。

  • 他在与印度教育家克里希那穆提 (Jiddu Krishnamurti) 的讨论中关注到思维本质是集体性而非个体性，思维本身存在自指悖论，传统内省自修无法真正理解心智，深层思维假定只能在与外界碰撞中显示出来；
  • 他与英国医生德玛赫 (Patrick de Mare) 关注到20人以上组成的群体可视为为整个文化系统的小型样本，包罗各式文化观念与价值体系，人们可以在长期对话中诞生彼此坦诚信任的「纯粹友谊」 (impersonal fellowship) ;
  • 1984年春，他注意到对话群体意义流动与不预设议程对话的重要意义：

周末时，大家都期待围绕对话内容开展一系列讲座，交流和研讨，最后发现对话本身其实涉及一些更重要的事情： 大家领悟到，对话的过程应当是让各种不同的意义在全体参与者之间自由流动。
在对话中，人们最初只是表达各自固定的立场，牢牢维护自己的见解； 随后他们意识到，维持群体友谊氛围比固守己见更重要。这从友谊是纯粹客观的，她的建立与对话者之间是否有密切私人关系无关。
由此而出现一种新的心态，这种心态立足于共同意义的建立与分享，而这种共同意义这在对话中不断更新改造。这时候人们不再对立，但也不能说达到了彼此交互作用的地步； 事实上他们开始一起参与并分享一个不断发展和变化的共同意义之库。在这一过程中，群体无需为对话预先设定任何目的，目的可以在对话的过程中随时自发生成与改变。群体成员之间由此进入一种新型动态关系之中，任何一个人都不会被拒绝发言，任何特定的对话内容都不会被排斥。
在上述原则指导下，我们就可以开始探索对话的种种可能性与潜力。照此坚持下去，我们更准可能去改造人与人之间的关系，甚至进而改造源生这些关系的意识之最终本质。

浪花拍向礁岩，碎裂，得见他人，重生自己。

个人思维紊乱分裂，失真截流，到世间繁事纠结矛盾，混乱险恶，如微分到积分，微雨纷纷落江海,不过累积放大。

对于自我，我们以思维观万物，却难反观思维本身。不自觉的社会观念与前置假定在左右我们，看似零散的不同行为往往是内隐思维问题的不同表现。保持长期对话会动摇我们所固守的文化观念，生存意义乃至自我认知。全新自我，真挚友谊随之生成，人与人间开放，共享，信任也成为可能。

人世问题复杂，答案也许难寻。但重要的是人们放弃自己固执的思维定势，超越人为的二元对立，兼容并蓄创造性解决问题。

孤塘自淤，必发臭腐烂而终； 唯大海无垠，方涌鲜活生命之可能。

浩浩汤汤，横无际涯。
